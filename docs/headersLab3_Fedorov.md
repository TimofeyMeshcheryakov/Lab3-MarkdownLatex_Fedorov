# Лаба номер № 3. работа с Markdown-разметкой и базовое использование LaTeX в документации проекта
## Markdown (маркдаун) — лёгкий язык разметки для текстовых документов. Создан Джоном Грубером и Аароном Шварцем в 2004 году. 
### Markdown находит широкое применение в различных сферах:
#### в разработке (GitHub, README-файлы), ведения заметок (Obsidian, Notion), документации, блогах (Jekyll), мессенджерах (Discord, Telegram), форумах (Stack Overflow) и для создания учебных материалов
##### Markdown нужен разработчикам для быстрого, удобного и универсального форматирования текстов без отвлечения на сложные редакторы
