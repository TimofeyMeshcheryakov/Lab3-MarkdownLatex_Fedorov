Markdown (маркдаун) — лёгкий язык разметки для текстовых документов. Создан Джоном Грубером и Аароном Шварцем в 2004 году. 
***
Markdown предоставляет возможности для быстрого и простого форматирования текста (заголовки, списки, ссылки, цитаты, жирный/курсив) с помощью легко запоминаемого синтаксиса, который остается читаемым даже в исходном виде.
***
Пример форматирования. Загловки:
# Заголовок 1
## Заголовок 2
### Заголовок 3
